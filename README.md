# BERD Academy Open Educational Resources

This site brings together materials from various sources to support learning, teaching, and exploring data-driven methods.

Workflow: When adding new material, please make sure that...
- ...the \_quarto.yml file is updated.
- ...the tags.yaml file is updated.
- ...the data type tags suit the headings in 01_data_types/00_intro.qmd.
- ...both the data type and the event tag links are updated in the 00_intro/01_how_to.qmd file.
